| File | Description |
|------|-------------|
| `Labyrinx.exe` | GUI — point-and-click obfuscation, EXE builder, and admin tools |

| `LABYRINX_USER_MANUAL.md` | Full user manual with CLI reference and troubleshooting |

## Quick Start

1. Download `Labyrinx.exe` — runs in **Freemium** mode (Level 1–2) for free
2. Subscribe to **Pro** ($9/mo) or **Enterprise** ($29/mo) to unlock higher levels
3. Enter your license key to activate

## Built-in Tools

The following are built from within `Labyrinx.exe` (Admin panel):
- **HWID Tool** — for hardware-locked license binding
- **License Creator** — for generating customer license keys

## System Requirements

- Windows 10 or later, 64-bit
- No Python installation required
