## Quick Start

1. Download `Labyrinx_Setup.exe` — runs in **Freemium** mode (Level 1–2) for free
2. Run the installer — no extraction needed
3. Subscribe to **Pro** ($9/mo) or **Enterprise** ($29/mo) to unlock higher levels
4. Enter your license key to activate

## Included Tools

The following are included with the installation:
- **Labyrinx.exe** — GUI for obfuscation, EXE building
- **labyrinx-cli.exe** — Command-line interface for CI/CD automation
- **get_hwid.exe** — Hardware ID tool for license binding
- **User Manual** — Full documentation

## System Requirements

- Windows 10 or later, 64-bit
- No Python installation required (embedded)
